# 재지정과 파이프 작업
- 파일 I/O 를 재지정
```
command 1> file # command 의 stdout을 파일로 재지정
command 2> file # command 의 stderr을 파일로 재지정
command &> file # command 의 stdout 과 stderr 을 모두 파일로 재지정
command >file 2>&1 # command 의 stdout 과 stderr 파일로 재지정
command > /dev/null # command 의 출력을 /dev/null 로 재지정하여 폐기
command < file # stdin 을 재지정
```

- command >1 file 예시
```
giri ~ %echo "test123" 1> temp.txt
giri ~ %bat temp.txt 
───────┬─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
       │ File: temp.txt
───────┼─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
   1   │ test123
───────┴─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
```

- command 2> file 예시
```
giri ~ %echo "test123" 2> temp.txt
test123
giri ~ %bat temp.txt
```
